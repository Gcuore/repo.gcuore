Gcuore AltStore Repository

How to Add This Repository

To add this repository to AltStore:

  1. Open AltStore on your iOS device
  2. Go to the "Browse" tab
  3. Tap the "+" button in the top-right corner
  4. Enter the following URL: https://gcuore.vercel.app/apps.json
  5. Tap "Add Source"
